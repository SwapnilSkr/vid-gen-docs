# Tech Stack

## Runtime and Framework

- **Bun** runtime for fast TypeScript execution
- **ElysiaJS** for HTTP routing and validation

## Database

- **MongoDB** with **Mongoose**
- Collections: templates, characters, compositions

## Media Processing

- **FFmpeg** via `fluent-ffmpeg`
- Aspect ratio conversion, overlays, audio mixing, subtitle burn-in

## AI and Speech

- **OpenRouter** for LLM script generation
- **ElevenLabs** for text-to-speech

## Storage

- **AWS S3** for all asset storage
- Local storage for temporary processing files

## Dev Tooling

- ESLint
- TypeScript
- Docker + docker-compose
