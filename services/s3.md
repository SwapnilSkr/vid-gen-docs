# S3 Storage

Implemented in `server/src/services/s3.service.ts`.

## Buckets and Folders

Objects are stored under the following logical folders:

- `templates/`
- `characters/`
- `compositions/` for composition output videos
- `reels/` for generated reel output videos
- `audio/`
- `subtitles/`

## Upload Helpers

- `uploadVideo(buffer, folder, originalFilename)`
- `uploadImage(buffer, folder, originalFilename)`
- `uploadAudio(buffer, originalFilename)`
- `uploadSubtitles(content, compositionId)`

File names are generated with a timestamp and random suffix via `generateFilename`.

## Deletion

`deleteFromS3(url)` parses the S3 key from a URL and deletes it.

## Presigned URLs

`getPresignedUrl` returns a temporary download URL for a stored object.
